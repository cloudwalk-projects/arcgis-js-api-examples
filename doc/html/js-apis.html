应用接口¶

通用规则¶

API 地址规则¶

规则: http://{api-server}/api/{api-name} {api-name} 为接口名称, 把中间的 . 替换成 / 即可.

比如 bigdb.static.search - 检索接口, 对应的地址为 http://{api-server}/api/bigdb/static/search

AccessToken 授权¶

部分接口访问需要相关的授权验证. 相关接口 参考 `connect.oauth.*`_

系统在 OAuth 2.0 接口验证授权后, 获得 AccessToken, 需要加在接口末尾.

比如 http://{api-server}/api/bigdb/static/search?accessToken={accessToken}

HTTP 请求方式¶

默认 HTTP 请求方式: POST

Content-Type 格式¶

请求内容如果是 JSON 格式的文本需要设置 Content-Type: application/json

错误信息格式¶

{
   "errcode": "1",
   "errmsg": "error msg"
}

errcode = 0, 默认情况下表示执行成功.

basemap¶

基本地图

create - 创建地图¶

输入参数

{
     // (必填)显示地图的容器标识
     divId:'map',
     // 中心点坐标
     center:[],
     // 默认缩放比例
     zoom:16,
     // 最小缩放比例
     minZoom:15,
     // 最大缩放比例
     maxZoom:19
 }

输出参数

esri.Map 对象

示例代码

require([
    "cw/basemap","cw/directions", ...
], function(basemap, directions, ... ) {

    var map = basemap.create({
        // 显示地图的容器标识
        divId:'map',
        // 中心点坐标
        center:[],
        // 默认缩放比例
        zoom:16,
        // 最小缩放比例
        minZoom:15,
        // 最大缩放比例
        maxZoom:19
    });
...
});

     // (必填)摄像头标识

 }

    directions.query({
        // 显示地图的容器标识
        map:map,
        // 摄像头信息
        cameras:[]
    });
...

经纬度信息 {x:x,y:x}

mercator2lonlat - 墨卡托坐标转经纬度坐标¶

输入参数

{
     // 经度
     x:0,
     // 维度
     y:0
 }

输出参数

经纬度信息 {x:x,y:x}

lonat - 获取地图经纬度¶

输入参数

evt 地图上的点击事件

输出参数

经纬度信息

{x:29.35, y:106.33}

示例代码

require([
    "cw/basemap", "cw/util", ...
], function(basemap, util, ... ) {

    var map = basemap.create({ divId:'map' });

    map.on('click', function(evt){

        var lonlat = util.lonlat(evt);

        // 输出经纬度信息
        console.log('lonlat(x:' + lonlat.x + ', y=' + lonlat.y + ')');
    });
...
});
