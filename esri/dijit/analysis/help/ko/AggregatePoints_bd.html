포인트 집약(Aggregate Points)

이 도구는 포인트 피처 레이어와 영역 피처 레이어에서 작동합니다. 입력 영역 피처는 폴리곤 레이어의 피처이거나 도구가 실행될 때 계산된 정사각형 또는 육각형 그룹일 수 있습니다. 이 도구는 먼저 지정된 각 영역에 속한 포인트를 확인합니다. 이 영역 내 포인트의 공간 관계를 확인한 후에는 영역 내 모든 포인트에 대한 통계가 계산되어 영역에 할당됩니다. 가장 기본적인 통계는 영역 내에 있는 포인트의 수이지만 다른 통계도 확인할 수 있습니다.

예를 들어 커피숍 위치에 대응하는 포인트 피처와 구/군에 대응하는 영역 피처가 있다고 가정하고 구/군별 커피 판매량을 요약해 보겠습니다. 커피숍에 TOTAL_SALES 속성이 있는 것으로 가정할 때 각 구/군 내의 TOTAL_SALES 총 합, 최소/최대 TOTAL_SALES 또는 개수, 범위, 표준편차, 분산 등의 기타 통계를 구할 수 있습니다.

이 도구는 시계열 데이터에도 사용할 수 있습니다. 입력 포인트에 시간이 활성화된 경우 시간 슬라이싱 옵션이 제공됩니다. 시간 슬라이싱을 사용하면 특정 시간 슬라이스를 살펴보면서 영역 내 포인트의 대응비를 계산할 수 있습니다. 예를 들어 매시간에 결과가 발생되는 시 간격을 살펴볼 수 있습니다.

시간 예로, 다양한 커피숍 위치에서 이루어진 각 트랜잭션에 대한 포인트 피처가 있지만 영역 레이어는 없다고 가정해 보겠습니다. 데이터는 1년 동안 기록되었으며 각 트랜잭션마다 위치와 시간 스탬프가 있습니다. 각 트랜잭션에 TOTAL_SALES 속성이 있는 것으로 가정할 경우 관심 시공간 내의 TOTAL_SALES에 대한 총 합을 구할 수 있습니다. 이러한 트랜잭션이 단일 도시에 대한 것이라면 1킬로미터 그리드의 영역을 생성하고 주단위 시간 슬라이스를 확인하여 시간과 공간 양쪽의 트랜잭션을 요약할 수 있습니다.

영역으로 집약할 포인트가 포함된 레이어 선택

영역으로 집약되는 포인트 레이어입니다.

맵에서 레이어를 선택하는 것 이외에, 드롭다운 목록의 하단에 있는 레이어 찾아보기를 선택하여 빅 데이터 파일 공유 데이터셋 또는 피처 레이어의 콘텐츠를 찾아볼 수 있습니다.

집약할 영역 레이어 선택

포인트 레이어가 집약되는 영역입니다. 제공한 영역 레이어 또는 도구 실행 시 생성되는 그룹 중에 선택할 수 있습니다. 정사각형 그룹 생성 시, 지정된 숫자와 단위에 따라 정사각형의 높이와 너비가 결정됩니다. 육각형 그룹의 경우 지정된 숫자와 단위는 평행한 두 변 간의 거리를 지정합니다.

집약할 영역 레이어 선택

포인트가 집약되는 영역 레이어입니다. 이 레이어의 영역 경계 내에 있는 모든 포인트 수가 계산되고 사용자가 선택한 통계를 사용하여 선택적으로 요약됩니다.

그룹의 그룹 크기 또는 영역 레이어를 설정해야 합니다.

그룹을 생성하고 집약할 거리 선택

입력 포인트가 집약될 그룹을 생성하는 데 사용되는 거리입니다. 정사각형 그룹의 경우 크기는 정사각형의 높이를 나타냅니다. 육각형 그룹의 경우 크기는 육각형의 높이(평행한 두 변 간의 거리)를 나타냅니다.

그룹의 그룹 크기 또는 영역 레이어를 설정해야 합니다.

시간 슬라이스를 사용하여 집약(선택)

입력 포인트 레이어에 시간이 활성화되었으며 시간 유형이 인스턴트인 경우 시간 슬라이싱을 사용하여 분석할 수 있습니다. 시간을 설정할 때 다음과 같은 세 가지 매개변수를 사용할 수 있습니다.

  • 시간 간격
  • 시간 단계
  • 참조 시간

시간에서 1년을 나타내는 데이터가 있는데 주 단위 슬라이스를 사용하여 분석한다고 가정해 보겠습니다. 이를 위해 시간 간격을 1주로 설정합니다.

시간에서 1년을 나타내는 데이터가 있는데 그 달의 첫 주를 사용하여 분석하고자 한다고 가정해 보겠습니다. 이를 위해서는 시간 간격을 1주로, 시간 단계를 1개월로, 참조 시간을 1월 1일 오전 12시로 설정합니다.

집약될 시간 간격

데이터 슬라이싱에 사용되는 시간 간격입니다. 시간 간격은 단독으로 사용하거나 시간 단계 또는 참조 시간 매개변수와 함께 사용할 수 있습니다.

매주 월요일 오전 9시부터 오전 10시까지 발생되는 시간 슬라이스를 생성하려고 했다면 시간 간격은 1시간, 시간 단계는 1주, 참조 시간은 월요일 오전 9시입니다.

집약될 시간 단계

시간 슬라이스를 계산하는 데 사용되는 단계입니다. 시간 단계는 단독으로 사용하거나, 시간 간격과 함께 사용하거나, 참조 시간과 함께 사용하거나, 시간 간격과 참조 시간 둘 다와 함께 사용할 수 있습니다.

매주 월요일 오전 9시부터 오전 10시까지 발생되는 시간 슬라이스를 생성하려고 했다면 시간 간격은 1시간, 시간 단계는 1주, 참조 시간은 월요일 오전 9시입니다.

시간 슬라이싱을 정렬하는 데 사용되는 참조 시간

시간 슬라이싱을 정렬하는 데 사용되는 날짜와 시간입니다. 시간 단계는 참조 시간을 기준으로 그 이전으로 진행됩니다.

매주 월요일 오전 9시부터 오전 10시까지 발생되는 시간 슬라이스를 생성하려고 했다면 시간 간격은 1시간, 시간 단계는 1주, 참조 시간은 월요일 오전 9시입니다.

통계 추가(선택)

요약된 피처에 대한 통계를 계산할 수 있습니다. 숫자 필드에 대해 다음을 계산할 수 있습니다.

  • 개수
  • 평균
  • 최소값
  • 최대값
  • 범위
  • 변수
  • 표준편차

문자열 필드에 대해 다음을 계산할 수 있습니다.

  • 개수
  • 임의 - 이 통계는 지정된 필드에 있는 문자열 값에 대한 랜덤 샘플입니다.
모든 통계는 null이 아닌 값에 대해 계산됩니다. 결과 레이어에는 계산된 각 통계에 대한 새 필드가 포함됩니다. 속성과 통계를 선택하여 원하는 개수의 통계를 추가할 수 있습니다.

SpatialReference(wkid)

이 매개변수는 프로세싱 공간 참조를 설정하기 위한 시험판용 임시 매개변수입니다. 대부분의 빅 데이터 도구의 경우 투영 좌표체계를 프로세싱 공간 참조로 사용해야 합니다. 기본 설정에 따라 이 도구는 입력 좌표체계를 사용하지만 지리 좌표체계일 경우에는 사용할 수 없습니다. 투영 좌표체계를 설정하려면 WKID를 입력합니다. 예를 들어 Web Mercator를 설정하려면 3857을 입력합니다.

데이터 저장소 선택

이 매개변수는 결과가 저장되는 데이터 저장소를 설정하기 위한 시험판용 임시 매개변수입니다. 두 데이터 저장소 모두 ArcGIS Data Store의 일부입니다. 시공간적 데이터 저장소를 선택했지만 설치되어 있지 않은 경우에는 도구를 사용할 수 없습니다.

결과 레이어 이름

내 콘텐츠에서 생성되어 맵에 추가되는 레이어의 이름입니다. 기본 이름은 도구 이름과 입력 레이어 이름을 기반으로 합니다. 레이어가 이미 있으면 도구를 사용할 수 없습니다.

결과 저장: 드롭다운 상자를 사용하여 내 콘텐츠에서 결과를 저장할 폴더의 이름을 지정할 수 있습니다.
