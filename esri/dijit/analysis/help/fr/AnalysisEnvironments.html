Paramètres de l'environnement d'analyse du portail

Paramètres de l'environnement d'analyse dans Portal for ArcGIS et ArcGIS Online.

Remplacer la couche de résultat

Option de remplacement des couches existantes. Lorsque cette option est sélectionnée, s'il existe une couche du même nom dans votre contenu, elle sera remplacée.

  • Cochée : s'il existe une couche du même nom dans votre contenu, elle sera remplacée. La sortie présentera le même système de coordonnées que les données en entrée.
  • Non cochée : si le nom de la couche de résultat existe déjà, une erreur se produit et la couche de résultat doit être renommée pour que l'outil puisse être exécuté. Il s'agit de l'option par défaut.

Fermer le widget d'analyse après l'exécution de l'analyse

Option permettant de garder la boîte de dialogue d'analyse ouverte et active pendant l'exécution de l'outil. Ce paramètre vous permet de modifier facilement vos paramètres d'analyse pendant que l'analyse précédente se poursuit. L'analyse est un processus itératif et en gardant la boîte de dialogue de l'outil ouverte, vous pouvez facilement modifier les paramètres et réexécuter votre analyse.

  • Cochée : la fenêtre de l'outil d'analyse se ferme et vous êtes redirigé vers la table des matières. Il s'agit du paramètre par défaut.
  • Non cochée : la fenêtre de l'outil d'analyse reste ouverte et active.

Stocker les résultats de l'analyse en tant que services d'entités

Option permettant d'indiquer si le résultat de votre analyse sera créé en tant que couche d'entités ou collecte d'entités.

  • Cochée : les résultats de votre analyse permettront de générer une couche d'entités et un élément sera créé dans votre contenu. Il s'agit de l'option par défaut.
  • Non cochée : les résultats de votre analyse vont générer une collecte d'entités. Les collectes d'entités sont stockées dans votre carte Web.

Système de coordonnées en sortie

Paramètre d'environnement pour l'analyse raster utilisé pour définir le système de coordonnées de la couche de résultat.

  • Identique à l'entrée : le résultat de votre analyse apparaîtra dans le même système de coordonnées que les données en entrée. Il s'agit de l'option par défaut.
  • Comme spécifié : permet de choisir le système de coordonnées des résultats de votre analyse. Lorsque cette option est sélectionnée, cliquez sur l'icône du globe à sélectionner dans une liste des systèmes de coordonnées connus ou ajoutez le WKID de référence spatiale dans la zone appropriée.
  • <Nom de la couche> : choisissez une couche de votre carte Web pour déterminer le système de coordonnées du résultat de votre analyse.

Traitement du système de coordonnées

Paramètre d'environnement de géoanalyse permettant de déterminer le système de coordonnées utilisé par l'analyse pendant son exécution.

Pour la géoanalyse, certaines analyses exigent que le traitement soit exécuté dans un système de coordonnées projetées. Tous les outils ayant classifié des données ou des unités linéaires nécessite un système de coordonnées projetées.

  • Identique à l'entrée : votre analyse sera traitée dans le même système de coordonnées que les données en entrée. Il s'agit de l'option par défaut.
  • Comme spécifié : permet de choisir le système de coordonnées qui traitera votre analyse. Lorsque cette option est sélectionnée, cliquez sur l'icône du globe à sélectionner dans une liste des systèmes de coordonnées projetées connus ou ajoutez le WKID de référence spatiale dans la zone appropriée.
  • Couche <nom> : choisissez une couche dans votre carte Web. Le système de coordonnées de la couche choisie permettra de déterminer le système de coordonnées qui sera utilisé pour traiter votre analyse.

Etendue

Paramètre d'environnement permettant de déterminer l'étendue ou la limite utilisée pendant l'exécution de l'analyse. Toutes les entités en entrée se trouvant intégralement dans l'étendue spécifiée ou qui intersectent cette dernière seront utilisées dans l'analyse.

  • Par défaut : utilise l'étendue fournie par l'outil.
  • Comme spécifié : indiquez manuellement les coordonnées pour définir l'étendue.
  • Couche <nom> : choisissez une couche dans votre carte Web. L'étendue spatiale de la couche choisie permettra de déterminer l'étendue utilisée pour traiter votre analyse.

Raster de capture

Lorsque vous procédez à une analyse raster, définissez l'environnement Raster de capture pour ajuster l'étendue de la couche raster en sortie afin qu'elle corresponde à l'alignement des cellules de la couche Raster de capture spécifiée.

Taille de cellule

Lorsque vous procédez à une analyse raster, cet environnement définit la taille de cellule ou la résolution qui sera utilisée pour créer la couche raster en sortie. La résolution en sortie par défaut est déterminée par la taille de cellule la plus importante dans la couche raster en entrée.

  • Minimum d’entrées : utilise la plus petite taille de cellule de toutes les couches en entrée.
  • Maximum d’entrées : utilise la plus grande taille de cellule de toutes les couches en entrée. Il s'agit de l'option par défaut.
  • Comme spécifié : indique une valeur numérique pour définir la taille de cellule. Si cette option est sélectionnée, la valeur par défaut est 1.
  • <Nom> de la couche : définit la taille de cellule sur la couche raster choisie.

Masque

Lorsque vous procédez à une analyse raster, utilisez l'environnement Masque pour spécifier une couche qui permettra de définir votre zone d'intérêt. Seules les cellules comprises dans le masque d'analyse seront prises en compte lors de l'analyse.

  • Le masque peut être un raster ou une couche d'entités.
  • Si le masque d'analyse est un raster, toutes les cellules ayant une valeur sont prises en compte pour définir le masque. Les cellules d'un raster de masque de type NoData seront considérées comme étant en dehors du masque et auront donc une valeur NoData dans la couche de résultat d'analyse.
  • Si le masque d'analyse est une couche d'entités, il est converti en interne en raster lors de l'exécution. Vous devez donc vérifier que la Taille de cellule et le Raster de capture sont correctement paramétrés pour votre analyse.
