Synthétiser les attributs

Cet outil synthétise toutes les valeurs d'appariement dans un ou plusieurs champs et calcule les statistiques les concernant. La statistique la plus élémentaire est le total d'entités qui ont été synthétisées, mais vous pouvez également calculer des statistiques plus avancées.

Supposons par exemple que des entités ponctuelles correspondent à l'emplacement de points de vente avec un champ représentant DISTRICT_MANAGER_NAME et que vous souhaitez synthétiser les ventes de café par responsable. Vous pouvez spécifier le champ DISTRICT_MANAGER_NAME comme champ de fusion afin que toutes les lignes de données représentant chaque responsable soient synthétisées. Cela signifie que tous les emplacements des points de vente gérés par Responsable1 seront synthétisés dans une seule ligne et que les résumés statistiques seront calculés. Dans cet exemple, les statistiques telles que le total de points de vente et la somme de TOTAL_SALES pour tous les points de vente gérés par Responsable1 sont calculées, ainsi que celles des autres responsables répertoriés dans le champ DISTRICT_MANAGER_NAME.

Choisir l'entrée à synthétiser

La couche contenant les champs qui seront synthétisés.

En plus de choisir une couche de votre carte, vous pouvez sélectionner Parcourir les couches au bas de la liste déroulante pour parcourir votre contenu et rechercher une couche d'entités ou un jeu de données de partage de fichiers Big Data.

Choisir les champs sur lesquels effectuer la fusion

Liste des champs de la couche en entrée. Les champs sélectionnés seront synthétisés et utilisés pour le calcul des statistiques. Par exemple, si vous avez choisi un champ nommé PropertyType avec les valeurs commerciales et résidentielles, tous les champs résidentiels sont synthétisés ensemble (les statistiques sont calculées) et tous les champs commerciaux sont synthétisés ensemble.

Choisir les champs et les statistiques à calculer (facultatif)

Vous pouvez calculer des statistiques sur des entités synthétisées. Dans les champs numériques, vous pouvez calculer ce qui suit :

  • Effectif
  • Moyenne
  • Min
  • Max.
  • Plage
  • Variance
  • Ecart type

Dans les champs de chaîne, vous pouvez calculer ce qui suit :

  • Effectif
  • Tout : cette statistique est un échantillon aléatoire d'une valeur de chaîne dans le champ spécifié.
Toutes les statistiques sont calculées sur des valeurs non nulles. La couche obtenue contient un nouveau champ pour chaque statistique calculée. Vous pouvez ajouter des statistiques en choisissant un attribut et une statistique.

Choisir un stockage des données

Il s'agit d'un paramètre temporaire de la version préliminaire qui permet de définir le stockage des données où les résultats sont enregistrés. Les deux stockages des données font partie d' ArcGIS Data Store. Si vous sélectionnez le stockage des données spatio-temporelles alors qu'aucun n'est installé, l'outil échoue.

Nom de la couche résultat

Il s'agit du nom de la couche qui sera créée dans votre page Mon contenu et ajoutée à la carte. Le nom par défaut repose sur le nom de l'outil et sur le nom de la couche en entrée. Si la couche existe déjà, l'outil échoue.

La zone de liste déroulante Enregistrer le résultat dans vous permet de spécifier le nom d'un dossier dans Mon contenu où le résultat sera enregistré.
