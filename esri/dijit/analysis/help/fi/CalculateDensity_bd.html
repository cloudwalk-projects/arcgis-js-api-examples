Laske tiheys

The Calculate Density tool creates a density map from point features by spreading known quantities of some phenomenon (represented as attributes of the points) across the map. The result is a layer of areas classified from least dense to most dense.

For point input, each point should represent the location of some event or incident, and the result layer represents a count of the incident per unit area. A higher density value in a new location means that there are more points near that location. In many cases, the result layer can be interpreted as a risk surface for future events. For example, if the input points represent locations of lightning strikes, the result layer can be interpreted as a risk surface for future lightning strikes.

Other use cases of this tool include the following:

  • Creating crime-density maps to help police departments properly allocate resources to high crime areas.
  • Calculating densities of hospitals within a county. The result layer will show areas with high and low accessibility to hospitals, and this information can be used to decide where new hospitals should be built.
  • Identifying areas that are at high risk of forest fires based on historical locations of forest fires.
  • Locating communities that are far from major highways in order to plan where new roads should be constructed.

Valitse pistekarttataso, joista tiheys lasketaan

The point layer on which the density will be calculated.

Sen lisäksi, että voit valita karttatason kartasta, voit valita Selaa karttatasoja -vaihtoehdon avattavan luettelon alareunasta ja selata Big Data -tiedostoresurssin aineistoryhmän tai karttatason sisältöön.

Valitse vähintään yksi kenttä, jonka tiheys lasketaan

Provide one or more fields specifying the number of incidents at each location. For example, if you have points that represent cities, you can use a field representing the population of the city as the count field, and the resulting population density layer will calculate larger population densities near cities with larger populations.

The density for the count of points will always be calculated. If no fields are selected, each location will be assumed to represent a single count.

Valitse lohkon muoto

The shape of bins that points are aggregated into and used for the density calculation.

Valitse lohkon koko tiheyslaskentaa varten

The size of the bins generated. For a square bin, the size represents the height of the square. For hexagon, the size represents the height of the hexagon (from parallel sides).

Valitse käytettävä tiheyden painotus

The type of weighting applied to the density calculation. There are two options:

  • Uniform—This calculates a magnitude per area.
  • Kernel—Applies a kernel function to fit a smooth tapered surface to each point.

Valitse naapuruston koko

The size of the area used to calculate the density. The neighborhood size must be larger than the bin size.

Valitse tiheyden tulosyksiköt

The desired output units of the density values. By default, this is set to the units in your map (for example, square meters). If density values are very small, you can increase the size of the area units (for example, square meters to square kilometers) to return larger values. The values only scale the result.

Laske tiheys käyttäen aikaleikkeitä (valinnainen)

Jos aika on otettu käyttöön lähtöaineiston karttatasossa ja jos ajan tyyppinä on hetki, voit käyttää analysointiin aikaleikkeitä. Voit määrittää kolme parametria, kun käytät aikaa:

  • Aikaväli
  • Aika-askel
  • Viiteaika

Oletetaan, että sinulla on vuotta kuvaava aineisto ja että haluat käyttää analysointiin aikaleikkeitä. Voit tehdä tämän määrittämällä Aikaväli-parametrin arvoksi 1 viikko.

Oletetaan, että sinulla on vuotta kuvaava aineisto ja että haluat käyttää analysointiin kuukauden ensimmäistä viikkoa. Voit tehdä tämän määrittämällä Aikaväli-parametrin arvoksi 1 viikko, Aika-askel-parametrin arvoksi 1 kuukausi ja Viiteaika-parametrin arvoksi 1. tammikuuta klo 24.00.

Aikaväli, jonka tiheys lasketaan

Aikaväli, jota käytetään aineiston leikkaamiseen. Aikaväliä voi käyttää yksin tai Aikaväli- tai Viiteaika-parametrin kanssa.

Jos haluat luoda aikaleikkeitä, joiden aikaväli on maanantaina klo 9.00–10.00, määritä Aikaväli-parametrin arvoksi 1 tunti, Aika-askel-parametrin arvoksi 1 viikko ja Viiteaika-parametrin arvoksi 9.00 maanantaina.

Aika-askel, jonka tiheys lasketaan

Askel, jota käytetään aikaleikkeen laskemiseen. Aika-askel-parametria voi käyttää yksin, Aikaväli-parametrin kanssa, Viiteaika-parametrin kanssa tai sekä Aikaväli- että Viiteaika-parametrin kanssa.

Jos haluat luoda aikaleikkeitä, joiden aikaväli on maanantaina klo 9.00–10.00, määritä Aikaväli-parametrin arvoksi 1 tunti, Aika-askel-parametrin arvoksi 1 viikko ja Viiteaika-parametrin arvoksi 9.00 maanantaina.

Viiteaika, jota käytetään aikaleikkeiden kohdistamiseen

Päivämäärä ja kellonaika, joita käytetään aikaleikkeiden kohdistamiseen. Aika-askel alkaa viiteajasta ja jatkuu viitatusta ajasta taaksepäin.

Jos haluat luoda aikaleikkeitä, joiden aikaväli on maanantaina klo 9.00–10.00, määritä Aikaväli-parametrin arvoksi 1 tunti, Aika-askel-parametrin arvoksi 1 viikko ja Viiteaika-parametrin arvoksi 9.00 maanantaina.

Koordinaatistotieto (wkid)

Tämä on esijulkaisun tilapäinen parametri käsittelyn koordinaatistotiedon määrittämiseen. Monet Big Data -työkalut edellyttävät, että projisoitua koordinaattijärjestelmää käytetään käsittelyn koordinaatistotietona. Oletusarvon mukaan työkalu käyttää lähtöaineiston koordinaattijärjestelmää, mutta työkalua ei voi suorittaa, jos kyseessä on maantieteellinen koordinaattijärjestelmä. Jos haluat määrittää projisoidun koordinaattijärjestelmän, anna WKID. Jos haluat esimerkiksi määrittää Web Mercator -koordinaattijärjestelmän, anna 3857.

Valitse aineistosäilö

Tämä on esijulkaisun tilapäinen parametri sen aineistosäilön määrittämiseen, johon tulokset tallennetaan. Molemmat aineistosäilöt ovat osa ArcGIS Data Storea. Jos valitset tila-aika-aineistosäilön, mutta sellaista ei ole asennettuna, työkalua ei voi suorittaa.

Tulostason nimi

Sen tason nimi, joka luodaan kohtaan Oma sisältö ja joka lisätään karttaan. Oletusnimi perustuu työkalun nimeen ja lähtökarttatason nimeen. Jos karttataso on jo olemassa, työkalua ei voi suorittaa.

Avattavan Tallenna tulos kohteeseen -valikon avulla voit määrittää sen Oma sisältö -kansion nimen, johon tulos tallennetaan.
