Convertir ráster a entidad

Esta herramienta convierte un ráster en un dataset de entidades como, por ejemplo, puntos, líneas o polígonos.

El parámetro Elegir campo le permite elegir qué campo de atributo del dataset ráster de entrada se convertirá en un atributo en la clase de entidad de salida. Este parámetro solo está disponible para rásteres enteros que tienen atributos adicionales en la tabla asociada.

El ráster de entrada puede tener cualquier tamaño de celda.

Los rásteres enteros se pueden convertir a entidades de punto, línea o polígono; no obstante, los rásteres de puntos flotante solo se pueden convertir a entidades de puntos.

Si la opción Utilizar la extensión de mapa actual está activada, solo se convertirán las celdas de ráster que estén visibles dentro de la extensión de mapa actual. Si la opción está desactivada, se convertirán todas las celdas de la capa ráster, aunque estén fuera de la extensión de mapa actual. Tenga en cuenta este aspecto si la capa ráster es grande o tiene una resolución muy alta.

Se describen a continuación ejemplos de uso de esta herramienta:

  • Crear entidades poligonales a partir de un ráster de clasificación de uso del suelo.
  • Los arroyos identificados en un DEM se pueden convertir a entidades de líneas.
  • Un DEM se puede convertir a entidades de puntos que se podrán utilizar en un flujo de trabajo no basado en ráster.

Elija una capa ráster para convertir

La capa ráster de entrada que se convertirá a capas de entidades.

Elegir campo

Proporcione un campo que especifique el valor que se utilizará para la conversión.

Puede ser cualquier valor entero o de texto.

Los campos que contienen valores de punto flotante solo se pueden utilizar si la salida es a un dataset de puntos.

La opción predeterminada consiste en utilizar el campo Valor, que contiene el valor de cada celda ráster.

Elegir tipo de salida

Elija el tipo de salida entre punto, línea o polígono.

  • Punto: el ráster se convertirá a un dataset de puntos.
  • Línea: el ráster se convertirá a un dataset de entidades de línea.
  • Polígono: el ráster se convertirá a un dataset de entidades poligonales.

Simplificar líneas o polígonos

Especifique si se deben simplificar las entidades.

Si el parámetro está activado (opción predeterminada), las entidades de línea o poligonales se suavizarán para generar un resultado más generalizado. Si esta opción está desactivada, las entidades lineales seguirán los límites de las celdas del dataset ráster.

Nombre de capa de resultados

El nombre de la capa que se creará en Mi contenido y que se agregará al mapa. El nombre predeterminado se basa en el nombre de la herramienta y en el nombre de la capa de entrada. Si la capa ya existe, se te pedirá que indiques otro nombre.

En el cuadro desplegable Guardar el resultado en, puedes especificar el nombre de una carpeta de Mi contenido en la que se guardará el resultado.
