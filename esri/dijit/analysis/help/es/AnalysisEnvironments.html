Configuración del entorno de análisis de Portal

Configuración del entorno para el análisis en Portal for ArcGIS y ArcGIS Online.

Sobrescribir capa de resultados

Opción para sobrescribir capas existentes. Cuando está activada, si ya existe una capa con el mismo nombre en el contenido, esta se sobrescribirá.

  • Activada: si ya existe una capa con el mismo nombre en el contenido, esta se sobrescribirá. La salida tendrá el mismo sistema de coordenadas que la entrada.
  • Desactivada: si el nombre de la capa de resultados indicado ya existe, se producirá un error y será necesario cambiar el nombre de la capa de resultados para poder ejecutar la herramienta. Esta es la opción predeterminada.

Cerrar widget de análisis después de ejecutar análisis

Opción para mantener el cuadro de diálogo de análisis abierto y habilitado mientras se procesa la herramienta. Este parámetro le permitirá realizar modificaciones fácilmente en los parámetros del análisis mientras el trabajo de análisis anterior se sigue ejecutando. El análisis es un proceso iterativo y, al mantener el cuadro de diálogo de la herramienta abierto, puede cambiar fácilmente parámetros y volver a ejecutar el análisis.

  • Activada: la ventana de la herramienta de análisis se cerrará y se volverá a la tabla de contenido. Esta es la configuración predeterminada
  • Desactivada: la ventana de la herramienta de análisis permanecerá abierta y habilitada.

Almacenar los resultados del análisis como servicios de entidades

Opción que permite especificar si el resultado del análisis se creará como una capa de entidades o como un conjunto de entidades.

  • Activada: los resultados del análisis se crearán como una capa de entidades y se creará un elemento en el contenido. Esta es la opción predeterminada.
  • Desactivada: los resultados del análisis crearán un conjunto de entidades. Los conjuntos de entidades se almacenan dentro del mapa web.

Sistema de coordenadas de salida

Configuración del entorno para el análisis de ráster utilizada para definir el sistema de coordenadas de la capa de resultados.

  • Como la entrada: el resultado del análisis estará en el mismo sistema de coordenadas que la entrada. Esta es la opción predeterminada.
  • Como se especifica: elija el sistema de coordenadas para los resultados del análisis. Si esta opción está seleccionada, haga clic en el icono de globo para seleccionar un valor de una lista de sistemas de coordenadas conocidos o agregue el WKID de la referencia especial en el cuadro correspondiente.
  • Nombre <de capa>: elija una capa existente del mapa web que se utilizará para determinar el sistema de coordenadas del resultado del análisis.

Procesando sistema de coordenadas

Configuración del entorno de geoanálisis utilizada para determinar el sistema de coordenadas que se utilizará en el análisis durante su ejecución.

Para el geoanálisis, en algunos análisis es necesario que el proceso se ejecute en un sistema de coordenadas proyectadas. Cualquier herramienta que haya implementado la colocación de bins de unidades de datos o lineales requiere un sistema de coordenadas proyectadas.

  • Como la entrada: el análisis se procesará en el mismo sistema de coordenadas que la entrada. Esta es la opción predeterminada.
  • Como se especifica: elija el sistema de coordenadas para procesar el análisis. Si esta opción está seleccionada, haga clic en el icono de globo para seleccionar un valor de una lista de sistemas de coordenadas proyectadas conocidos o agregue el WKID de la referencia especial en el cuadro correspondiente.
  • Nombre <de capa>: elija una capa existente del mapa web. El sistema de coordenadas de la capa elegido se utilizará para determinar el sistema de coordenadas que se utilizará para procesar el análisis.

Extensión

Configuración del entorno utilizada para determinar la extensión o el límite que se utilizará durante la ejecución del análisis. En el análisis se utilizarán todas las entidades de entrada que estén completamente dentro de la extensión especificada o que la intersequen.

  • Valor predeterminado: se utiliza la extensión proporcionada por la herramienta.
  • Como se especifica: introduzca manualmente las coordenadas para definir la extensión.
  • Nombre <de capa>: elija una capa existente del mapa web. La extensión espacial de la capa elegida se utilizará para determinar la extensión utilizada para procesar el análisis.

Alinear ráster

Al realizar el análisis de ráster, si se define el entorno de Alinear ráster, se ajustará la extensión de la capa ráster de salida para que coincida con la alineación de celda de la capa de Alinear ráster especificada.

Tamaño de Celda

Al realizar el análisis de ráster, este entorno define el tamaño o la resolución de celda que se utilizará para crear la capa ráster de salida. La resolución de salida predeterminada se determina según el tamaño de celda más grande de la capa ráster de entrada.

  • Cantidad mínima de entradas: utiliza el tamaño de celda más pequeño de todas las capas de entrada.
  • Cantidad máxima de entradas: utiliza el tamaño de celda más grande de todas las capas de entrada. Esta es la opción predeterminada.
  • Como se especifica: especifique un valor numérico para definir el tamaño de celda. Si se selecciona esta opción, el valor predeterminado es 1.
  • Nombre <de capa>: define el tamaño de celda según la capa ráster elegida.

Máscara

Al realizar el análisis de ráster, utilice el entorno de la Máscara para especificar una capa que se utilizará para definir el área de interés. Solo se incluirán en la operación de análisis aquellas celdas que queden dentro de la máscara de análisis.

  • La máscara puede ser un ráster o una capa de entidades.
  • Si la máscara de análisis es un ráster, se considerarán todas las celdas que tienen un valor para definir la máscara. Las celdas en un ráster de máscara que son NoData se considerarán fuera de la máscara y serán NoData en la capa de resultados del análisis.
  • Si la máscara de análisis es una capa de entidades, se convertirá internamente en un ráster en la ejecución. Por este motivo, asegúrese de que Tamaño de celda y Alinear ráster se establezcan adecuadamente para el análisis.
