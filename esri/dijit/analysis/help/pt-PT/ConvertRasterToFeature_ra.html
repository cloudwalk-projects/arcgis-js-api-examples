Converter Raster para Elemento

Esta ferramenta converte um raster para um conjunto de dados de elementos, em forma de pontos, linhas ou polígonos.

O parâmetro Escolher campo permite-lhe selecionar o campo de atributos no qual o conjunto de dados raster de entrada se tornará um atributo na classe de elementos de saída. Este parâmetro encontra-se disponível apenas para rasters inteiros que tenham atributos adicionais na tabela associada.

O raster de entrada pode ter qualquer tamanho de célula.

Rasters inteiros podem ser convertidos para elementos de ponto, linha ou polígono; no entanto, rasters de ponto flutuante podem apenas ser convertidos para elementos de ponto.

Se a opção Utilizar a extensão atual do mapa estiver marcada , serão convertidas apenas as células raster que se encontrem visíveis na atual extensão do mapa. Se não marcar a opção, serão convertidas todas as células da camada raster, mesmo que se encontrem fora da extensão atual do mapa. Tome esta informação em conta no caso de a camada raster ser de grande dimensão ou ter uma resolução muito elevada.

Outros casos de utilização desta ferramenta incluem os seguintes:

  • Criar elementos de polígono a partir de um raster de classificação de utilização do solo.
  • Riachos identificados numa DEM podem ser convertidos em elementos de linha.
  • Uma DEM pode ser convertida em elementos de ponto que podem ser utilizados em fluxos de trabalho não baseados em raster.

Selecionar camada raster para converter

A camada raster de entrada que será convertida em camadas de elementos.

Escolher campo

Introduza um campo que especifique que valor será utilizado para a conversão.

Pode ser um valor inteiro ou de texto.

Um campo que contenha valores de ponto flutuantes pode ser utilizado apenas se a saída for para um conjunto de dados de ponto.

A escolha predefinida consiste em utilizar o campo Valor , que contém o valor de cada célula raster.

Escolher tipo de saída

Escolha o tipo de saída de entre ponto, linha ou polígono.

  • Ponto—O raster será convertido para um conjunto de dados de ponto.
  • Linha—O raster será convertido para um conjunto de dados de elementos de linha.
  • Polígono—O raster será convertido para um conjunto de dados de elementos de polígono.

Simplificar linhas ou polígonos

Especifique se os elementos devem ser simplificados.

Se estiver marcada (a predefinição), os elementos de linha ou de polígono serão suavizados, por forma a produzirem um resultado mais generalizado. Caso esta opção se encontre desmarcada, os elementos lineares seguirão os limites das células do conjunto de dados raster.

Nome da camada resultante

O nome da camada que será criada em O Meu Conteúdo e adicionada ao mapa. O nome padrão é baseado no nome da ferramenta e do nome da camada de entrada. Se a camada já existir, ser-lhe-á pedido que forneça outro nome.

Ao utilizar a caixa suspensa Guardar resultado em pode definir o nome da pasta em O Meu Conteúdo onde o resultado será guardado.
