Copiar para Armazenamento de Dados

Esta ferramenta irá copiar uma camada de entrada para um armazenamento de dados. Os dados serão copiados para o ArcGIS Data Store e serão armazenados no seu armazenamento de dados relacional ou espaciotemporal.

Por exemplo, poderia copiar elementos que se encontram armazenados numa partilha de dados big data para um armazenamento de dados relaciona e especificar que penas elementos incluídos na atual extensão de mapa serão copiados. Isto criaria um serviço de elementos alojados apenas com os elementos que se encontravam no interior da extensão de mapa definida.

Escolher camada a copiar

A camada ou tabela que será copiada.

Para além de selecionar uma camada do seu mapa, pode selecionar Explorar Camadas na parte inferior da lista pendente para navegar até aos seus conteúdos de um conjunto de dados de partilha de big data ou camada de elementos.

O armazenamento de dados para o qual a camada será copiada. As opções são as seguintes:

  • Espaciotemporal—Esta é a predefinição.
  • Relacional.

Nome da camada resultante

Este é o nome da camada que será criada em O Meu Conteúdo e adicionada ao mapa. O nome padrão é baseado no nome da ferramenta e do nome da camada de entrada. Caso a camada já existe, a ferramenta irá falhar.

Ao utilizar a caixa suspensa Guardar resultado em pode definir o nome da pasta em O Meu Conteúdo onde o resultado será guardado.
