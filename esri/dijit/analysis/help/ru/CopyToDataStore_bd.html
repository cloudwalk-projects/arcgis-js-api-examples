Копировать в хранилище данных

Инструмент копирует входной слой в хранилище данных. Данные копируются в ArcGIS Data Store и будут храниться в реляционном или пространственно-временном хранилище данных.

Например, вы можете скопировать объекты, хранящиеся в файловом хранилище больших данных в реляционное хранилище, при этом скопировав только объекты в пределах текущего экстента карты. При этом будет создан размещенный сервис объектов, содержащий объекты, которые находились в пределах текущего экстента карты.

Выбрать слой для копирования

Копируемый слой или таблица.

Кроме выбора слоя из вашей карты, можно щелкнуть Выбрать слои в нижней части ниспадающего списка, чтобы найти свои ресурсы для набора данных или слоя объектов файлового хранилища больших данных .

Выбор хранилища ArcGIS Data Store для копирования

Хранилище данных, в которое будет скопирован слой. Используются следующие опции:

  • Пространственно-временное – используется по умолчанию.
  • Реляционное.

Имя слоя результата

Это имя слоя, который будет создан в разделе Мои ресурсы и добавлен на карту. Имя слоя по умолчанию зависит от имени инструмента и имени входного слоя. Если слой уже существует, произойдет сбой.

С помощью ниспадающего списка поля Сохранить результат в вы можете задать имя папки в разделе Мои ресурсы, в которую будет записан результат.
