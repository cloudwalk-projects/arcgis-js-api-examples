Суммировать атрибуты

Этот инструмент суммирует все совпадающие значения одного или нескольких полей и вычисляет для них статистику. Самая простая статистика - количество суммированных объектов, но вычислить можно и более сложную.

Например, у вас есть точечные объекты, соответствующие местоположениям магазинов, содержащие поле DISTRICT_MANAGER_NAME, и вы хотите определить объемы продаж кофе по менеджерам. Можно указать поле DISTRICT_MANAGER_NAME в качестве поля слияния, и тогда будут суммированы все строки для каждого из менеджеров. Это означает, что все местоположения магазинов, управляемых Manager1 будут суммированы в одну строку, для которой выполнится вычисление суммарной статистики. В данном случае такая статистика, как число магазинов и сумма значений TOTAL_SALES будет вычислена для всех магазинов, управляемых Manager1, точно так же, как и всеми остальными менеджерами, перечисленными в поле DISTRICT_MANAGER_NAME.

Выбрать входные данные для суммирования

Слой, содержащий суммируемые поля.

Кроме выбора слоя из вашей карты, можно щелкнуть Выбрать слои в нижней части ниспадающего списка, чтобы найти свои ресурсы для набора данных или слоя объектов файлового хранилища больших данных .

Выбрать поля, с которыми хотите выполнить слияние

Список полей входного слоя. Выбранные поля будут суммированы и использованы для вычисления статистики. К примеру, если вы выберете поле PropertyType, содержащее принадлежность недвижимости к коммерческой или жилой, будут суммированы отдельно все поля как для жилой, так и для коммерческой недвижимости с вычисленной статистикой.

Выбрать поля и статистику, которые хотите вычислить (дополнительно)

Вы можете вычислить статистику по суммируемым объектам. По числовым полям можно вычислить:

  • Количество
  • Среднее
  • Мин.
  • Макс.
  • Диапазон
  • Дисперсия
  • Среднеквадратическое отклонение

По строковым полям можно вычислить:

  • Количество
  • Любая – эта статистика является случайной выборкой строкового значения в указанном поле.
Вся статистика вычисляется на ненулевых значениях. Итоговый слой будет содержать новое поле для каждой вычисленной статистики. Можно добавить любое число вычисленных статистических значений, выбрав атрибут и статистику.

Выбрать хранилище данных

Это временный параметр предварительной версии для задания хранилища данных для сохранения результатов. Оба хранилища данных являются частью ArcGIS Data Store. Если вы выбрали хранилище пространственно-временных данных, но оно еще не установлено, произойдет сбой.

Имя слоя результата

Это имя слоя, который будет создан в разделе Мои ресурсы и добавлен на карту. Имя слоя по умолчанию зависит от имени инструмента и имени входного слоя. Если слой уже существует, произойдет сбой.

С помощью ниспадающего списка поля Сохранить результат в вы можете задать имя папки в разделе Мои ресурсы, в которую будет записан результат.
