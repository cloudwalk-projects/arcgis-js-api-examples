Resumir Atributos

Esta ferramenta resume todos os valores de comparação em um ou mais campos e calcula as estatísticas deles. A estatística mais básica é a contagem de feições que foi resumida, mas você pode calcular estatísticas mais avançadas também.

Por exemplo, suponha que você tenha feições de ponto de locais de armazenamento com um campo representando o DISTRICT_MANAGER_NAME e você deseja resumir as vendas de café por gerente. Você pode especificar o campo DISTRICT_MANAGER_NAME como o campo para dissolver e todas as linhas de dados representando os gerentes individuais que serão resumidos. Isto significa que todos locais de armazenamento que são gerenciados por Manager1 serão resumidos em uma linha com estatísticas sumárias calculadas. Nesta instância, a estatística como contagem do número de lojas e a soma de TOTAL_SALES para todas as lojas que o Manager1 gerencia seriam calculadas como também para qualquer outro gerente listado no campo DISTRICT_MANAGER_NAME.

Escolha a entrada para resumir

A camada que contém os campos que serão resumidos.

Além de escolher uma camada de seu mapa, você pode escolher Procurar Camadas na parte inferior da lista suspensa para procurar por seu conteúdo para um conjunto de dados de compartilhamento de arquivo big data ou camada de feição.

Escolha os campos nos quais você deseja dissolver

Lista de campos a partir da camada de entrada. Os campos selecionados serão resumidos e utilizados para calcular estatísticas. Por exemplo, se você escolheu um campo denominado PropertyType com os valores de comerciais e residenciais, todos os campos residenciais seriam resumidos, com estatística sumária calculada e todos os campos comerciais seriam resumidos.

Escolha os campos e estatísticas que você deseja calcular (opcional)

Você pode calcular as estatística em feições que são resumidas. Nos campos numéricos você pode calcular o seguinte:

  • Contagem
  • Médio
  • Mín
  • Máx
  • Intervalo
  • Variância
  • Desvio Padrão

Nos campos de string você pode calcular o seguinte:

  • Contagem
  • Qualquer—Esta estatística é uma amostra aleatória de um valor de string no campo especificado.
Todas as estatísticas são calculadas em valores não nulos. A camada resultante conterá um novo atributo para cada estatística calculada. Qualquer número de estatística pode ser adicionado ao escolher um atributo e estatística.

Escolher armazenamento de dados

Este é um parâmetro temporário para pré lançamento para configurar o armazenamento de dados onde os resultados são salvos. Ambos os armazenamentos de dados são parte do ArcGIS Data Store. Se você selecionar o armazenamento de dados spatiotemporal mas não ter um instalado, a ferramenta falhará.

Nome da camada resultante

Este é o nome da camada que será criada em Meu Conteúdo e adicionada ao mapa. O nome padrão é baseado no nome da ferramenta e o nome da camada de entrada. Se a camada já existe, a ferramenta falhará.

Ao utilizar a caixa suspensa Salvar resultado em, é possível especificar o nome de uma pasta em Meu Conteúdo onde o resultado será salvo.
