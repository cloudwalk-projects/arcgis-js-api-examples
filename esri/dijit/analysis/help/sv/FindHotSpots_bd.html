Hitta hotspots

Med verktyget Hitta hotspots kan du fastställa om det finns statistiskt signifikanta kluster i det geografiska mönstret för dina data.

  • Är dina punkter (brottsfall, träd eller trafikolyckor) verkligen klustrade? Hur kan du vara säker på det?
  • Har du verkligen hittat en statistiskt signifikant hotspot? Eller skulle kartan berätta en annan historia om du ändrade hur den symboliserades?
Verktyget Hitta hotspots hjälper dig besvara de här frågorna med säkerhet.

Till och med slumpmässiga geografiska mönster uppvisar någon grad av klustring. Det är dessutom naturligt för våra ögon och hjärnor att försöka hitta mönster även när det inte finns några. Därför kan det vara svårt att veta om mönstren i dina data är resultatet av verkliga geografiska processer eller slumpen. Detta är orsaken till att forskare och analytiker använder statistiska metoder som Hitta hotspots (Getis-Ord Gi*) för att mäta geografiska mönster. När du hittar statistiskt signifikant klustring i dina data har du värdefull information. Om du vet var och när klustring uppstår kan det ge viktiga ledtrådar om processerna som leder till de mönster du ser. Kunskapen om att till exempel bostadsinbrott är betydligt högre i vissa grannskap kan vara avgörande för att ta fram effektiva förebyggande strategier, fördela polisens knappa resurser, starta bevakningsprogram, bemyndiga större polisundersökningar eller identifiera möjliga gärningsmän.

Välj det lager som hotspots ska beräknas för

Det punktlager som hotspots och coldspots ska hittas i.

Förutom att välja ett lager från kartan kan du välja Bläddra bland lager längst ned i listrutan för att bläddra till ditt innehåll efter ett big data-fildelningsdataset eller geoobjektslager.

Hitta kluster med hög och låg

Den här analysen besvarar frågan: Var klustras höga och låga värden geografiskt?

Om dina data utgörs av punkter och du väljer Punktantal, analyserar verktyget den geografiska ordningen av punktgeoobjekten för att besvara frågan: Var har punkterna oväntat bildat kluster eller spridits ut?

Om du väljer ett fält analyserar verktyget den geografiska ordningen för värden som kopplas till varje geoobjekt för att besvara frågan: Var har de höga respektive låga värdena bildat kluster?

Välj lagerplatsstorlek för hotspotberäkningar

Det avstånd som används för att skapa lagringsplatserna som ska användas för att analysera dina indatapunkter.

Aggregera med tidsdelar (valfritt)

Om tid är aktiverat på punktlagret och det är av typen tidspunkt, kan du analysera med tidsintervall.

Tidsintervall som ska aggregeras till

Tidsintervall som används för analys. Om du vill skapa tidsdelar som inträffar varje vecka ställer du in tidsintervallet till en vecka.

Tidssteg som ska aggregeras till

Hur tiden ska justeras. Tiden kan justeras till start- eller sluttiden för indata eller till en angiven referenstid.

Referenstid som används till att justera tidsdelar

Datum och tid som används för att justera tidsintervallen.

Välj närområdets storlek

Avståndet som används för att avgöra det närområde som används för hotspotberäkningar. Närområdet bör vara större än lagringsplatsstorleken för att se till att varje lagringsplats har minst en granne. Varje lagringsplats analyseras och jämförs med angränsande lagringsplatser.

SpatialReference (wkid)

Det här är en tillfällig parameter för förhandsversionen för att ställa in den geografiska referensen för bearbetning. Många big data-verktyg kräver att ett projicerat koordinatsystem används som geografisk referens för bearbetning. Som standard använder verktyget indatakoordinatsystemet men misslyckas om det är ett geografiskt koordinatsystem. Ange WKID för att ställa in ett projicerat koordinatsystem. Till exempel anges Web Mercator som 3857.

Välj datalager

Det här är en tillfällig parameter för förhandsversionen för att ange det datalager där resultaten sparas. Båda datalagren ingår i ArcGIS Data Store. Om du väljer det rumstemporala datalagret men inte har ett installerat misslyckas verktyget.

Resultatlagrets namn

Det här är namnet på lagret som skapas i Mitt innehåll och läggs till i kartan. Standardnamnet baseras på verktygets namn och indatalagrets namn. Om lagret redan finns misslyckas verktyget.

Resultatlagret visar statistiskt signifikanta kluster med höga och låga värden eller punktantal. Om lagrets namn redan finns uppmanas du att byta namn på det.

Om du använder listrutan Spara resultat i kan du ange namnet på en mapp i Mitt innehåll där resultatet ska sparas.
