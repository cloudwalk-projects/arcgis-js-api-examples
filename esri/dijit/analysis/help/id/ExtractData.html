Ekstrak Data

Dengan alat ini, Anda dapat memilih dan mengunduh data untuk area acuan tertentu. Layer yang Anda pilih akan ditambahkan ke file ZIP, CSV, atau KMZ.

Layer untuk Mengekstrak

Ada beberapa layer dalam daftar isi Anda yang dapat digunakan untuk mengekstrak fitur. Layer-layer yang dicentang adalah layer yang diekstrak dan termasuk pada file hasil yang berupa ZIP atau CSV.

Agar layer-layer tersebut muncul pada daftar layer yang akan diekstrak, Anda harus merupakan pemilik data atau pemilik data tersebut harus mengaktifkan fitur ekspor data. Untuk mengaktifkan fitur ekspor data pada data yang Anda miliki:

  • Buka item data deskripsi
  • Klik tab Pengaturan.
  • Gulir ke bawah ke bagian Export Data (Ekspor Data)
  • Centang Izinkan orang lain mengekspor ke format berbeda. Klik Simpan saat selesai.

Jika layer analisis Anda mengandung lampiran, perhatikan bahwa lampiran tersebut hanya akan diekstrak jika format data output adalah file geodatabase dan Clip fitur tidak dicentang.

Area studi

Area yang menentukan fitur mana yang akan dimasukkan sebagai file output ZIP atau CSV. Anda dapat memilih dari yang berikut ini:

  • Jangkauan visibel peta Anda saat ini ( Sama seperti Tampilan).
  • Satu opsi membatasi jumlah maksimum lokasi terdekat untuk ditemukan per lokasi mulai.
  • Waktu berkendara mengikuti pilihan Anda terkait kecepatan perjalanan, baik yang berdasarkan lalu lintas sesungguhnya saat itu atau situasi lalu lintas pada umumnya.
Semua fitur yang berada dalam jangkauan peta akan diekstrak. Untuk fitur-fitur yang bersilangan atau memotong batas jangkauan peta, namun tidak sepenuhnya berada dalam jangkauan peta, Anda memiliki dua opsi:
  • Pilih fitur—Seluruh fitur yang melewati batas area studi akan diekstrak.
  • Clip fitur—Hanya bagian fitur yang berada dalam area studi yang akan diekstrak.

Format data output

Format data yang akan diekstrak. Saat mengekspor fitur garis atau area ke file CSV, hanya atribut nonspasial yang akan diekstrak. Saat mengekspor fitur titik, baik atribut spasial maupun nonspasial akan diekstrak.

Nama file output

Ini adalah nama item yang akan ditambahkan ke Konten Saya Anda. Nama item akan dipilih berdasarkan dengan Format data output yang dipilih. Jika item sudah ada, Anda akan diminta untuk memberikan nama lain.

Menggunakan kotak tarik turun (drop-down box) Simpan hasil di, Anda dapat menentukan nama folder di Konten Saya tempat hasilnya akan disimpan.
