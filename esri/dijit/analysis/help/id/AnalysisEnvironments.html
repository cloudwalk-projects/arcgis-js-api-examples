Analisis Portal Pengaturan Lingkungan

Pengaturan lingkungan untuk analisis pada Portal untuk ArcGIS dan ArcGIS Online.

Timpa layer hasil

Opsi untuk menimpa layer yang ada. Saat dicentang, jika suatu layer dengan nama sama telah ada pada konten Anda, layer itu akan ditimpa.

  • Dicentang—jika suatu layer dengan nama sama telah ada pada konten Anda, layer itu akan ditimpa. Output akan memiliki sistem koordinat yang sama dengan inputnya.
  • Tidak dicentang—Jika nama layer hasil yang diberikan telah ada, kesalahan akan terjadi dan layer hasil harus diberi nama kembali sebelum alat dapat dieksekusi. Ini adalah default.

Tutup widget analisis setelah menjalankan analisis

Opsi untuk menyimpan dialog analisis terbuka dan aktif saat alat memproses. Pengaturan ini akan memungkinkan Anda secara mudah melakukan modifikasi pada parameter analisis Anda saat tugas analisis sebelumnya terus berjalan. Analisis adalah proses yang berulang dan dengan menjaga dialog alat terbuka, itu memungkinkan Anda untuk dengan mudah melakukan pengubahan parameter dan menjalankan kembali analisis Anda.

  • Dicentang—Jendela alat analisis akan menutup dan fokus akan kembali ke daftar isi. Ini adalah default
  • Tidak dicentang—Jendela alat analisis akan tetap terbuka dan aktif.

Simpan hasil analisis sebagai feature service

Opsi untuk menentukan apakah hasil analisis Anda akan dibuat sebagai feature layer atau sebagai kumpulan fitur.

  • Dicentang—Hasil analisis Anda akan dibuat feature layer dan item akan dibuat pada konten Anda. Ini adalah default.
  • Tidak Dicentang—Hasil analisis Anda akan membuat kumpulan fitur. Kumpulan fitur disimpan pada web map Anda.

Sistem koordinat output

Pengaturan lingkungan untuk analisis raster digunakan untuk menentukan sistem koordinat layer hasil.

  • Sama seperti Input—Hasil analisis Anda akan berada di sistem koordinat yang sama dengan input. Ini adalah default.
  • Seperti yang ditentukan—Pilih sistem koordinat untuk hasil analisis Anda. Ketika opsi ini dipilih, klik ikon bola dunia untuk memilih dari daftar sistem koordinat yang diketahui atau menambahkan referensi spasial WKID pada kotak yang disediakan.
  • Layer <nama>—Pilih layer yang sudah ada pada web map Anda yang akan digunakan untuk menentukan sistem koordinat hasil analisis Anda.

Memproses sistem koordinat

Pengaturan lingkungan untuk GeoAnalytics digunakan untuk menentukan sistem koordinat yang akan analisis Anda gunakan saat eksekusi analisis.

Untuk GeoAnalytics, beberapa analisis mengharuskan bahwa proses dieksekusi pada sistem koordinat terproyeksi. Setiap alat yang menerapkan penempatan data atau unit linear mengharuskan sistem koordinat terproyeksi.

  • Sama seperti input—Analisis Anda akan diproses pada sistem koordinat yang sama seperti input. Ini adalah default.
  • Seperti yang ditentukan—Pilih sistem koordinat untuk memproses analisis Anda. Ketika opsi ini dipilih, klik ikon bola dunia untuk memilih dari daftar sistem koordinat terproyeksi yang diketahui atau menambahkan referensi spasial WKID pada kotak yang disediakan.
  • Layer <nama>—Pilih layer yang ada dari web map Anda. Sistem koordinat layer yang dipilih akan digunakan untuk menentukan sistem koordinat yang akan digunakan untuk memproses analisis Anda.

Jangkauan

Pengaturan lingkungan digunakan untuk menentukan jangkauan atau batasan yang akan digunakan saat eksekusi analisis. Seluruh fitur input yang sepenuhnya di dalam atau yang berpotongan dengan jangkauan tertentu akan digunakan pada analisis.

  • Default—Menggunakan jangkauan yang disediakan oleh alat.
  • Seperti yang ditentukan—Secara manual masukkan koordinat untuk menentukan jangkauan.
  • Layer <nama>—Pilih layer yang ada dari web map Anda. Jangkauan spasial layer yang dipilih akan digunakan untuk menentukan jangkauan yang digunakan untuk memproses analisis Anda.

Snap Raster

Saat melakukan analisis raster, mengatur lingkungan Snap Raster akan menyesuaikan jangkauan dari output raster layer sehingga itu sesuai dengan perataan sel layer Snap Raster yang ditentukan.

Ukuran Sel

Saat melakukan analisis raster, lingkungan ini mengatur ukuran sel atau resolusi yang akan digunakan untuk membuat output raster layer. Resolusi output default ditentukan oleh ukuran sel terbesar dari input raster layer.

  • Input Minimum—Gunakan ukuran sel terkecil dari seluruh layer input.
  • Input Maksimum—Gunakan ukuran sel terbesar dari seluruh layer input. Ini adalah default.
  • Seperti yang ditentukan—Tentukan nilai angka untuk menetapkan ukuran sel. Jika ini dipilih, nilai default adalah 1.
  • Layer <nama>—Menentukan ukuran sel ke raster layer yang dipilih.

Mask

Saat melakukan analisis raster, gunakan lingkungan Mask untuk menentukan layer yang akan digunakan untuk menentukan area penting Anda. Hanya sel tersebut yang berada dalam mask analisis yang akan diperhitungkan dalam operasi analisis.

  • Mask dapat berupa raster atau feature layer.
  • Jika mask analisis adalah raster, seluruh sel yang memiliki nilai akan diperhitungkan untuk menentukan mask. Sel pada raster mask yang merupakan NoData akan dianggap berada di luar mask dan akan menjadi NoData pada layer hasil analisis.
  • Jika mask analisis adalah feature layer, mask akan diubah secara internal menjadi raster saat eksekusi. Oleh karena itu, Anda harus hati-hati untuk memastikan bahwa Ukuran Sel dan Snap Raster diatur sesuai untuk analisis Anda.
