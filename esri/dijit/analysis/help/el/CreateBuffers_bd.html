Δημιουργία Ζωνών

Η ζώνη είναι μια περιοχή που καλύπτει μια δεδομένη απόσταση από ένα σημειακό, γραμμικό ή πολυγωνικό στοιχείο.

Οι ζώνες συνήθως χρησιμοποιούνται για τη δημιουργία περιοχών, οι οποίες μπορούν να αναλυθούν περισσότερο με χρήση άλλων εργαλείων. Για παράδειγμα, αν η ερώτηση είναι "Ποια κτίρια υπάρχουν σε απόσταση 1 μιλίου από το σχολείο;", η απάντηση μπορεί να βρεθεί, αν δημιουργηθεί μια ζώνη 1 μιλίου γύρω από το σχολείο και επικαλυφθεί η ζώνη με το θεματικό επίπεδο που περιέχει τα ίχνη των κτιρίων. Το τελικό αποτέλεσμα είναι ένα θεματικό επίπεδο με τα κτίρια που βρίσκονται σε απόσταση 1 μιλίου από το σχολείο.

Διαλέξτε θεματικό επίπεδο για εφαρμογή ζώνης

Τα σημειακά, γραμμικά ή πολυγωνικά στοιχεία για τα οποία θα δημιουργηθεί ζώνη.

Εκτός από την επιλογή ενός θεματικού επιπέδου από το χάρτη σας, μπορείτε επίσης να επιλέξετε Αναζήτηση θεματικών επιπέδων στο κάτω μέρος της αναπτυσσόμενης λίστας για αναζήτηση στα περιεχόμενά σας ενός κοινόχρηστου αρχείου dataset με big data ή ενός feature layer.

Επιλέξτε τον τύπο της ζώνης προς εφαρμογή

Υπάρχουν τρεις τρόποι με τους οποίους μπορείτε να προσδιορίσετε το μέγεθος της ζώνης για τα στοιχεία εισόδου:

  • Προσδιορίστε μια απόσταση που θα εφαρμοστεί σε όλα τα στοιχεία.
  • Προσδιορίστε ένα πεδίο στο θεματικό επίπεδο εισόδου το οποίο περιέχει την τιμή απόστασης. Μπορείτε να χρησιμοποιήσετε ένα πεδίο συμβολοσειράς ή ένα αριθμητικό πεδίο. Αν δεν οριστεί γραμμική μονάδα, θα χρησιμοποιηθούν οι μονάδες της χωρικής αναφοράς. Αν χρησιμοποιείτε σύστημα γεωγραφικών συντεταγμένων, το πεδίο χωρίς μονάδες θα θεωρηθεί ότι είναι μέτρα.
  • Δημιουργήστε μια έκφραση με πολλαπλά πεδία και μαθηματικούς τελεστές.

Επιλέξτε τη μέθοδο ζώνης

Μπορείτε να επιλέξετε να χρησιμοποιήσετε μια Επιπεδική ή Γεωδαιτική μέθοδο. Η Επιπεδική μέθοδος μπορεί να είναι ταχύτερη και είναι κατάλληλη για τοπική ανάλυση σε προβολικά δεδομένα. Η Γεωδαιτική μέθοδος είναι κατάλληλη για μεγάλες περιοχές και οποιοδήποτε σύστημα γεωγραφικών συντεταγμένων.

Επιλέξτε τον τύπο της μεθόδου ομαδοποίησης

Επιλογές για τον προσδιορισμό της μεθόδου ομαδοποίησης. Αν επιλεχθεί μια μέθοδος ομαδοποίησης, θα έχετε τη δυνατότητα να δημιουργήσετε πολυτμηματικές ή μονοτμηματικές περιοχές και θα μπορείτε να υπολογίσετε στατιστικά με βάση τα παρεχόμενα πεδία.

  • Όλα—Θα ομαδοποιηθούν όλα τα στοιχεία. Αν επιτρέπονται πολυτμηματικά στοιχεία, όλα τα στοιχεία θα ομαδοποιηθούν σε ένα και μόνο στοιχείο. Αν δεν επιτρέπονται πολυτμηματικά στοιχεία, θα ομαδοποιηθούν τα γειτονικά ή τα επικαλυπτόμενα στοιχεία.
  • Πεδία—Θα ομαδοποιηθούν τα στοιχεία που μοιράζονται το ίδιο καθορισμένο πεδίο ή τον καθορισμένο συνδυασμό πεδίων. Αν επιτρέπονται πολυτμηματικά στοιχεία, όλα τα στοιχεία με το ίδιο πεδίο θα ομαδοποιηθούν σε ένα και μόνο στοιχείο. Αν δεν επιτρέπονται πολυτμηματικά στοιχεία, θα ομαδοποιηθούν τα γειτονικά ή τα επικαλυπτόμενα στοιχεία με την ίδια τιμή πεδίου.
  • Κανένα—Κανένα στοιχείο δεν θα ομαδοποιηθεί. Αυτή είναι η προεπιλογή.

Να επιτρέπονται πολυτμηματικά στοιχεία

Επιλογή που προσδιορίζει εάν τα αποτελέσματά σας θα περιλαμβάνουν μονοτμηματικά ή πολυτμηματικά στοιχεία.

  • Επιλεγμένη—Τα αποτελέσματα της ανάλυσής σας θα περιέχουν πολυτμηματικά στοιχεία.
  • Μη επιλεγμένη—Τα αποτελέσματα της ανάλυσής σας θα περιέχουν μονοτμηματικά στοιχεία. Αυτή είναι η προεπιλογή.

Προσθήκη στατιστικών (προαιρετικά)

Μπορείτε να υπολογίσετε στατιστικά για τα στοιχεία που θα συνοψιστούν. Σε αριθμητικά πεδία, μπορείτε να υπολογίσετε τα εξής:

  • Πλήθος
  • Μέση τιμή
  • Ελάχ.
  • Μέγ.
  • Εύρος
  • Διακύμανση
  • Τυπική απόκλιση

Σε πεδία συμβολοσειράς, μπορείτε να υπολογίσετε τα εξής:

  • Πλήθος
  • Οποιαδήποτε—Αυτό το στατιστικό στοιχείο είναι ένα τυχαίο δείγμα μιας τιμής συμβολοσειράς στο καθορισμένο πεδίο.
Όλα τα στατιστικά στοιχεία υπολογίζονται σε τιμές που δεν είναι null. Το εξαγόμενο θεματικό επίπεδο θα περιέχει ένα νέο πεδίο για κάθε υπολογιζόμενο στατιστικό στοιχείο. Μπορεί να προστεθεί οποιοσδήποτε αριθμός στατιστικών στοιχείων, επιλέγοντας γνώρισμα και στατιστικό στοιχείο.

Επιλογή χώρου αποθήκευσης δεδομένων

Είναι μια προσωρινή παράμετρος για την έκδοση prerelease που ορίζει το data store αποθήκευσης των αποτελεσμάτων. Και οι δύο χώροι αποθήκευσης αποτελούν τμήμα του ArcGIS Data Store. Αν επιλέξετε το χωρο-χρονικό data store αλλά δεν υπάρχει εγκατεστημένο, το εργαλείο θα αποτύχει.

Όνομα εξαγόμενου θεματικού επιπέδου

Αυτό είναι το όνομα του θεματικού επιπέδου που θα δημιουργηθεί στην ενότητα Το περιεχόμενό μου και θα προστεθεί στο χάρτη. Το προεπιλεγμένο όνομα βασίζεται στο όνομα του εργαλείου και στο όνομα του θεματικού επιπέδου εισόδου. Αν το θεματικό επίπεδο υπάρχει ήδη, το εργαλείο θα αποτύχει.

Με τη χρήση του πτυσσόμενου πλαισίου Αποθήκευση αποτελέσματος σε, μπορείτε να καθορίσετε το όνομα ενός φακέλου στην ενότητα Το περιεχόμενό μου όπου θα αποθηκευτεί το αποτέλεσμα.
