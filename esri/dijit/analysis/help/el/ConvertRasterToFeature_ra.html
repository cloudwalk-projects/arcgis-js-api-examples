Μετατροπή raster σε στοιχείο

Το εργαλείο μετατρέπει ένα raster σε σύνολο δεδομένων στοιχείων, όπως σημεία, γραμμές ή πολύγωνα.

Η παράμετρος Επιλέξτε πεδίο σάς επιτρέπει να επιλέξετε το πεδίο γνωρίσματος του συνόλου raster δεδομένων εισόδου που θα μετατραπεί σε γνώρισμα στην κλάση στοιχείων εξόδου. Αυτή η παράμετρος είναι διαθέσιμη μόνο για raster ακεραίου που έχουν πρόσθετα γνωρίσματα στο συσχετισμένο πίνακα.

Το raster εισόδου μπορεί να έχει οποιοδήποτε μέγεθος κελιού.

Τα raster ακεραίου μπορούν να μετατραπούν σε σημειακά, γραμμικά ή πολυγωνικά στοιχεία. Ωστόσο, τα raster κινητής υποδιαστολής μπορούν να μετατραπούν μόνο σε σημειακά στοιχεία.

Αν είναι επιλεγμένη η Χρήση τρέχουσας έκτασης χάρτη, θα μετατραπούν μόνο τα κελιά του raster που είναι ορατά στην τρέχουσα έκταση του χάρτη. Αν δεν είναι επιλεγμένη, θα μετατραπούν όλα τα κελιά στο θεματικό επίπεδο raster, ακόμη και αν βρίσκονται έξω από την τρέχουσα έκταση του χάρτη. Πρέπει να το λάβετε υπόψη εάν το θεματικό επίπεδο raster είναι μεγάλο ή έχει πολύ υψηλή ανάλυση.

Μερικές περιπτώσεις χρήσης αυτού του εργαλείου είναι οι εξής:

  • Δημιουργία πολυγωνικών στοιχείων από raster ταξινόμησης χρήσης γης.
  • Οι ροές που εντοπίζονται σε ένα ψηφιακό μοντέλο εδάφους (DEM) μπορούν να μετατραπούν σε γραμμικά στοιχεία.
  • Ένα ψηφιακό μοντέλο εδάφους (DEM) μπορεί να μετατραπεί σε σημειακά στοιχεία που μπορούν να χρησιμοποιηθούν σε μια ροή εργασιών που δεν βασίζεται σε raster.

Επιλέξτε θεματικό επίπεδο raster για μετατροπή

Το raster θεματικό επίπεδο εισόδου που θα μετατραπεί σε feature layer.

Επιλέξτε πεδίο

Επιλέξτε ένα πεδίο που προσδιορίζει την τιμή που θα χρησιμοποιηθεί για τη μετατροπή.

Η τιμή μπορεί να είναι ακέραιος ή κείμενο.

Ένα πεδίο που περιέχει τιμές κινητής υποδιαστολής μπορεί να χρησιμοποιηθεί μόνο εάν η έξοδος παραπέμπει σε σημειακό σύνολο δεδομένων.

Η προεπιλεγμένη επιλογή είναι να χρησιμοποιηθεί το πεδίο Τιμή, το οποίο περιέχει την τιμή σε κάθε κελί του raster.

Επιλέξτε τύπο εξόδου

Επιλέξτε τον τύπο εξόδου ανάμεσα σε σημείο, γραμμή ή πολύγωνο.

  • Σημείο—Το raster θα μετατραπεί σε σύνολο σημειακών δεδομένων.
  • Γραμμή—Το raster θα μετατραπεί σε σύνολο γραμμικών δεδομένων.
  • Πολύγωνο—Το raster θα μετατραπεί σε σύνολο πολυγωνικών δεδομένων.

Απλοποίηση γραμμών ή πολυγώνων

Προσδιορίστε εάν πρέπει να απλοποιηθούν τα στοιχεία.

Αν επιλεχθεί (προεπιλογή), τα γραμμικά ή πολυγωνικά στοιχεία θα εξομαλυνθούν ώστε να εξαχθεί ένα πιο γενικευμένο αποτέλεσμα. Αν δεν επιλεχθεί, τα γραμμικά στοιχεία θα ακολουθήσουν τα όρια κελιών του συνόλου raster δεδομένων.

Όνομα εξαγόμενου θεματικού επιπέδου

Το όνομα του θεματικού επιπέδου που θα δημιουργηθεί στην ενότητα Το περιεχόμενό μου και θα προστεθεί στο χάρτη. Το προεπιλεγμένο όνομα βασίζεται στο όνομα του εργαλείου και στο όνομα του θεματικού επιπέδου εισόδου. Αν το θεματικό επίπεδο υπάρχει ήδη, θα σας ζητηθεί να δώσετε άλλο όνομα.

Με τη χρήση του πτυσσόμενου πλαισίου Αποθήκευση αποτελέσματος σε, μπορείτε να καθορίσετε το όνομα ενός φακέλου στην ενότητα Το περιεχόμενό μου όπου θα αποθηκευτεί το αποτέλεσμα.
