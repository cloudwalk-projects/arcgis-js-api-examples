Attribute zusammenfassen

Mit diesem Werkzeug werden alle übereinstimmenden Werte in einem oder mehreren Feldern zusammengefasst und Statistiken für sie berechnet. Die einfachste Statistik ist die Anzahl der zusammengefassten Features, Sie können aber auch erweiterte Statistiken berechnen.

Angenommen, Sie verfügen über Punkt-Features von Ladenstandorten mit einem Feld, das den DISTRICT_MANAGER_NAME darstellt, und Sie möchten die Umsatzzahlen für Kaffee nach Manager zusammenfassen. Sie können das Feld DISTRICT_MANAGER_NAME als Feld für die Zusammenführung auswählen, sodass alle Datenzeilen, die einzelne Manager darstellen, zusammengefasst werden. Dies bedeutet, dass alle Ladenstandorte, die von Manager1 geleitet werden, in einer Zeile mit berechneter Summenstatistik zusammengefasst werden. In diesem Fall werden Statistiken wie die Anzahl der Läden und die Summe der TOTAL_SALES für alle Läden, die von Manager1 geleitet werden, und für alle anderen Manager berechnet, die im Feld DISTRICT_MANAGER_NAME aufgeführt sind.

Die zusammenfassende Eingabe auswählen

Der Layer enthält die Felder, die zusammengefasst werden.

Sie können nicht nur einen Layer aus der Karte auswählen, sondern auch unten in der Dropdown-Liste Layer durchsuchen auswählen, um zu Ihren für ein Big-Data-Dateifreigabe-Dataset bzw. -Feature-Layer zu navigieren.

Die Felder für die Zusammenführung auswählen

Liste der Felder aus dem Eingabe-Layer. Ausgewählte Felder werden zusammengefasst und zum Berechnen von Statistiken verwendet. Wenn Sie beispielsweise ein Feld namens PropertyType mit den Werten von Gewerbe- oder Wohngebäuden ausgewählt haben, werden alle Wohngebäudefelder mit berechneten Summenstatistiken und alle Gewerbefelder zusammengefasst.

Die Felder und Statistiken auswählen, die berechnet werden sollen (optional)

Sie können Statistiken für Features berechnen, die zusammengefasst wurden. Für numerische Felder kann Folgendes berechnet werden:

  • Anzahl
  • Mittelwert
  • Min.
  • Max.
  • Bereich
  • Varianz
  • Standardabweichung

Für Zeichenfolgenfelder kann Folgendes berechnet werden:

  • Anzahl
  • Beliebig: Diese Statistik ist eine auf dem Zufallsprinzip basierende Stichprobe eines Zeichenfolgenwertes im angegebenen Feld.
Alle Statistiken werden für Nicht-NULL-Werte berechnet. Der resultierende Layer enthält ein neues Feld für jede berechnete Statistik. Jede Anzahl von Statistiken kann durch Auswahl eines Attributs und einer Statistik ausgewählt werden.

Data Store auswählen

Dies ist ein temporärer Parameter für die Vorabversion zum Festlegen des Data Store, in dem die Ergebnisse gespeichert werden. Beide Data Stores sind Teil des ArcGIS Data Store. Wenn Sie den Data Store "spatiotemporal" auswählen, aber nicht installiert haben, kann das Werkzeug nicht ausgeführt werden.

Name des Ergebnis-Layers

Dies ist der Name des Layers, der in Eigene Inhalte erstellt und der Karte hinzugefügt wird. Der Standardname basiert auf dem Werkzeugnamen und dem Namen des Eingabe-Layers. Wenn der Layer bereits vorhanden ist, kann das Werkzeug nicht ausgeführt werden.

Über das Dropdown-Feld Ergebnis speichern in können Sie den Namen eines Ordners in Eigene Inhalte angeben, in dem das Ergebnis gespeichert werden soll.
