Raster extrahieren

Dieses Werkzeug kann ein Raster auf eine Grenze ausschneiden; entweder auf eine rechteckige Fläche oder auf eine Form, die interaktiv auf dem Bildschirm definiert wird. Sie können das Raster auf die Ausdehnung der Fläche ausschneiden, die aktuell auf der Karte angezeigt wird, oder auf das Untersuchungsgebiet, das durch ein Polygon definiert wird.

Ausschneidegeometrie festlegen

Wählen Sie den Ausdehnungstyp aus, der beim Ausschneiden der Geometrie des Layers verwendet werden soll:

  • Aktuelle Ausdehnung verwenden: Die aktuelle Kartenausdehnung dient zum Ausschneiden des resultierenden Layers. Dies ist die Standardeinstellung.
  • Untersuchungsgebiet auswählen: Mit dieser Option können Sie festlegen, ob das Ergebnis die Fläche innerhalb oder die Fläche außerhalb der Geometrieausdehnung anzeigen soll. Wenn Sie die Option Daten innerhalb der Geometrie auswählen, werden die Bilddaten außerhalb der Ausdehnungen entfernt. Wenn Sie die Option Daten außerhalb der Geometrie auswählen, werden die Bilddaten innerhalb der Ausdehnungen entfernt. Eine Geometrie zum Ausschneiden kann auch mit dem Zeichenwerkzeug definiert werden.

Name des Ergebnis-Layers

Der Name des Layers, der in Eigene Inhalte erstellt und der Karte hinzugefügt wird. Der Standardname basiert auf dem Werkzeugnamen und dem Namen des Eingabe-Layers. Wenn der Layer bereits vorhanden ist, werden Sie aufgefordert, einen anderen Namen einzugeben.

Über das Dropdown-Feld Ergebnis speichern in können Sie den Namen eines Ordners in Eigene Inhalte angeben, in dem das Ergebnis gespeichert werden soll.
