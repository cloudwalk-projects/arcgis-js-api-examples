Raster in Feature konvertieren

Dieses Werkzeug konvertiert ein Raster in ein Feature-Dataset, wie Punkte, Linien oder Polygone.

Mithilfe des Parameters Feld auswählen können Sie festlegen, welches Attributfeld des Eingabe-Raster-Datasets in der Ausgabe-Feature-Class zu einem Attribut wird. Dieser Parameter ist nur für ganzzahlige Raster verfügbar, deren verknüpfte Tabelle zusätzliche Attribute aufweist.

Das Eingabe-Raster kann eine beliebige Zellengröße haben.

Ganzzahlige Raster können in Punkt-, Linien oder Polygon-Features konvertiert werden, Gleitkomma-Raster lassen sich jedoch nur in Punkt-Features konvertieren.

Wenn die Option Aktuelle Kartenausdehnung verwenden aktiviert ist, werden nur die Raster-Zellen konvertiert, die in der aktuellen Kartenausdehnung sichtbar sind. Wenn diese Option deaktiviert ist, werden auch dann alle Raster-Layer konvertiert, wenn sie sich außerhalb der aktuellen Kartenausdehnung befinden. Beachten Sie dies, wenn der Raster-Layer groß ist oder eine sehr hohe Auflösung hat.

Nachfolgend finden Sie Anwendungsfälle dieses Werkzeugs:

  • Erstellen Sie Polygon-Features aus einem Landnutzungsklassifizierungs-Raster.
  • In einem DEM identifizierte Wasserläufe können in Linien-Features konvertiert werden.
  • Ein DEM kann in Punkt-Features konvertiert werden, die in einem nicht Raster-basierten Workflow verwendet werden können.

Zu konvertierenden Raster-Layer auswählen

Der Eingabe-Raster-Layer, der in Feature-Layer konvertiert wird.

Feld auswählen

Legen Sie ein Feld fest, das angibt, welcher Wert für die Konvertierung verwendet werden soll.

Dies kann ein ganzzahliger oder ein Textwert sein.

Ein Feld, das Gleitkommawerte enthält, kann nur verwendet werden, wenn die Ausgabe in einem Punkt-Dataset erfolgt.

Standardmäßig wird das Feld Wert verwendet, das den Wert in jeder Raster-Zelle enthält.

Ausgabetyp auswählen

Wählen Sie den Ausgabetyp zwischen Punkt, Linie oder Polygon aus.

  • Punkt: Das Raster wird in ein Punkt-Dataset konvertiert.
  • Linie: Das Raster wird in ein Linien-Feature-Dataset konvertiert.
  • Polygon: Das Raster wird in ein Polygon-Feature-Dataset konvertiert.

Linien oder Polygone vereinfachen

Legen Sie fest, ob die Features vereinfacht werden sollen.

Wenn diese Option aktiviert ist (Standardeinstellung), werden die Linien- oder Polygon-Features geglättet, um ein generalisierteres Ergebnis zu erstellen. Wenn diese Option deaktiviert ist, folgen die linearen Features den Zellengrenzen des Raster-Datasets.

Name des Ergebnis-Layers

Der Name des Layers, der in Eigene Inhalte erstellt und der Karte hinzugefügt wird. Der Standardname basiert auf dem Werkzeugnamen und dem Namen des Eingabe-Layers. Wenn der Layer bereits vorhanden ist, werden Sie aufgefordert, einen anderen Namen einzugeben.

Über das Dropdown-Feld Ergebnis speichern in können Sie den Namen eines Ordners in Eigene Inhalte angeben, in dem das Ergebnis gespeichert werden soll.
