Impostazioni d'ambiente per l'analisi in Portal

Impostazioni d'ambiente per l'analisi in Portal for ArcGIS e ArcGIS Online.

Sovrascrivi layer risultati

Opzione per sovrascrivere i layer esistenti. Quando l'opzione è selezionata, se nei contenuti personali esiste già un layer con lo stesso nome, questo verrà sovrascritto.

  • Selezionata: se nei contenuti personali esiste già un layer con lo stesso nome, questo verrà sovrascritto. L'output avrà lo stesso sistema di coordinate dell'input.
  • Deselezionata: se esiste già il nome del layer dei risultati specificato, si verificherà un errore e il layer dei risultati dovrà essere rinominato prima che lo strumento venga eseguito. Si tratta dell'impostazione predefinita.

Chiudi widget di analisi dopo l'esecuzione dell'analisi

Opzione per mantenere la finestra di dialogo dell'analisi aperta e abilitata mentre lo strumento è in esecuzione. Questa impostazione consentirà di apportare facilmente modifiche ai parametri di analisi mentre continua l'esecuzione del lavoro di analisi precedente. L'analisi è un processo iterativo e, mantenendo aperta la finestra di dialogo, consente di modificare facilmente i parametri ed eseguire di nuovo l'analisi.

  • Selezionata: la finestra dello strumento di analisi verrà chiusa e verrà di nuovo visualizzato il sommario. Si tratta dell'impostazione predefinita.
  • Deselezionata: la finestra dello strumento di analisi rimarrà aperta e abilitata.

Memorizza risultati di analisi come feature service

Opzione per specificare se i risultati dell'analisi verranno creati come feature layer o come raccolta di feature.

  • Selezionata: i risultati dell'analisi verranno creati come feature layer e verrà creato un elemento nei contenuti personali. Si tratta dell'impostazione predefinita.
  • Deselezionata: i risultati dell'analisi verranno creati come raccolta di feature. Le raccolte di feature sono memorizzate all'interno della mappa Web.

Sistema di coordinate di output

Impostazione d'ambiente per analisi raster utilizzata per definire il sistema di coordinate del layer dei risultati.

  • Lo stesso come input: il risultato dell'analisi avrà lo stesso sistema di coordinate dell'input. Si tratta dell'impostazione predefinita.
  • Come specificato: scegliere il sistema di coordinate per i risultati dell'analisi. Quando questa opzione è selezionata, fare clic sull'icona del globo per selezionare da un elenco di sistemi di coordinate noti o aggiungere il WKID del riferimento spaziale nella casella fornita.
  • Layer <nome>: scegliere un layer esistente nella mappa Web che verrà utilizzata per determinare il sistema di coordinate del risultato dell'analisi.

Elaborazione del sistema di coordinate

Impostazione d'ambiente per GeoAnalytics utilizzata per determinare il sistema di coordinate che verrà utilizzato durante l'esecuzione dell'analisi.

Per GeoAnalytics, alcune analisi richiedono che l'elaborazione venga eseguita in un sistema di coordinate proiettato. Tutti gli strumenti che hanno implementato il binning dei dati o delle unità lineari richiedono un sistema di coordinate proiettato.

  • Lo stesso come input: l'analisi verrà elaborata nello stesso sistema di coordinate dell'input. Si tratta dell'impostazione predefinita.
  • Come specificato: scegliere il sistema di coordinate per elaborare l'analisi. Quando questa opzione è selezionata, fare clic sull'icona del globo per selezionare da un elenco di sistemi di coordinate proiettati noti o aggiungere il WKID del riferimento spaziale nella casella fornita.
  • Layer <nome>: scegliere un layer esistente dalla mappa Web. Il sistema di coordinate del layer scelto servirà per determinare il sistema di coordinate che verrà utilizzato per elaborare l'analisi.

Estensione

Impostazione d'ambiente utilizzata per determinare l'estensione o il confine che verrà utilizzato durante l'esecuzione dell'analisi. Tutte le feature di input che sono completamente all'interno o che intersecano l'estensione specificata verranno utilizzate nell'analisi.

  • Predefinito: utilizza l'estensione fornita dallo strumento.
  • Come specificato: immettere manualmente le coordinate per definire l'estensione.
  • Layer <nome>: scegliere un layer esistente dalla mappa Web. L'estensione spaziale del layer scelto servirà per determinare l'estensione che verrà utilizzata per elaborare l'analisi.

Snap raster

Quando si esegue l'analisi raster, l'impostazione d'ambiente Snap raster regolerà l'estensione del layer raster di output in modo che corrisponda all'allineamento cella del layer Snap raster specificato.

Dimensione cella

Quando si esegue l'analisi raster, questa impostazione d'ambiente definisce la dimensione della cella o la risoluzione che verrà utilizzata per creare il layer raster di output. La risoluzione di output predefinita è determinata dalla dimensione di cella più grande del layer raster di input.

  • Numero minimo di input: utilizzare la dimensione cella più piccola di tutti i layer di input.
  • Numero massimo di input: utilizzare la dimensione cella più grande di tutti i layer di input. Si tratta dell'impostazione predefinita.
  • Come specificato: specificare un valore numerico per impostare la dimensione della cella. Se questa opzione è selezionata, il valore predefinito è 1.
  • Layer <nome>: imposta la dimensione della cella sul layer raster scelto.

Maschera

Quando si esegue l'analisi raster, utilizzare l'impostazione d'ambiente Maschera per specificare un layer che verrà utilizzato per definire l'area di interesse. Solo le celle che rientrano nella maschera di analisi verranno considerate nell'operazione di analisi.

  • La maschera può essere un raster o un feature layer.
  • Se la maschera di analisi è un raster, tutte le celle contenenti un valore verranno considerate per definire la maschera. Le celle in un raster maschera che sono NoData verranno considerate esterne alla maschera e saranno NoData nel layer dei risultati dell'analisi.
  • Se la maschera dell'analisi è un feature layer, sarà convertito internamente in un raster al momento dell'esecuzione. Per questo motivo, assicurarsi che Dimensioni cella e Snap raster siano impostati appropriatamente per l'analisi.
