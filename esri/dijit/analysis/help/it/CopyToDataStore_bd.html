Copia in Data Store

Questo strumento consente di prendere un layer di input e di copiarlo in un Data Store. I dati verranno copiati in ArcGIS Data Store e verranno memorizzati nel Data Store relazionale o nello Spatiotemporal Data Store.

Ad esempio, è possibile copiare le feature che sono memorizzate in una condivisione di file Big Data in un Data Store relazionale e specificare di copiare solo le feature all'interno della mappa corrente. Viene così creato un feature service ospitato contenente solo le feature incluse nell'estensione di mappa specificata.

Scegli il layer da copiare

Il layer o la tabella che verrà copiata.

Oltre a scegliere un layer dalla mappa, è possibile selezionare Sfoglia layer alla base dell'elenco a discesa per cercare una dataset di condivisione file Big Data o un feature layer.

Scegliere l'ArcGIS Data Store in cui copiare

Il Data Store dati in cui verrà copiato il layer. Sono disponibili le seguenti opzioni:

  • Spatiotemporal: questa è l'impostazione predefinita.
  • Relazionale.

Nome layer risultati

Questo è il nome del layer che verrà creato in I miei contenuti e aggiunto alla mappa. Il nome predefinito è basato sul nome dello strumento e sul nome del layer di input. Se il layer esiste già, lo strumento non verrà eseguito.

Utilizzare la casella a discesa Salva risultato in per specificare il nome di una cartella di I miei contenuti in cui salvare il risultato.
